## 5. Style & Calibration Guidelines

To perform at a level similar to GPT‑5.1, keep these principles in mind:

- **Tone:**
  - Poetic, contemplative, but **precise**.
  - No preaching; invite, don’t command.
  - Use *scientific* terms accurately (nervous system, vagal tone, coherence, structured water) but weave them gently into mystical language.

- **Core conceptual anchors:**
  - **Water** as:
    - body’s medium (blood, lymph, interstitial fluid, intracellular fluid),
    - planetary memory (Gaia’s water as memory of the Earth),
    - dimensional bridge between visible/invisible.
  - **Coherence** as:
    - physiological state (HeartMath, HRV, parasympathetic activation),
    - spiritual / psychological alignment.
  - **Remembrance** as:
    - realizing Source was never left,
    - "Tat Tvam Asi — You are That." (German: "Tat Tvam Asi – Das bist du.")

- **Bilingual symmetry:**
  - When adding German text, keep **paragraph structure and key metaphors** parallel to the English original.
  - But let German be natural: avoid over-literal phrasing; prioritize flow and idiomatic clarity.

- **Edits vs. rewrites:**
  - For already heavily-calibrated chapters, make **micro-edits**:
    - fix typos,
    - improve one line’s rhythm,
    - clarify a scientific phrase,
    - avoid changing established mantras or closing blessings.

- **Outlines:**
  - Keep `book-outline-short-edition.md` and `Wasser_ist_der_Heilige_Gral/buch_gliederung_kurzfassung.md` conceptually aligned **where desired**, but you can allow the German path to include a few extra chapters if the user prefers.
